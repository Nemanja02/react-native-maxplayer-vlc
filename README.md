# react-native-maxplayer-vlc

This package is a **forked version** of the original *react-native-vlc-media-player* library.  
Original project:  
https://github.com/razorRun/react-native-vlc-media-player.git

This fork introduces extended functionality, improved stability, and additional commands specifically adapted for the MaxPlayer project.

## Installation

```bash
npm install react-native-maxplayer-vlc
# or
yarn add react-native-maxplayer-vlc
